# e-plantShopping

This repository contains the e-plantShopping React application built for the Paradise Nursery project.  
The project includes a landing page, About Us page, product listings grouped by category, and a fully functional shopping cart implemented with Redux Toolkit.

## Live Deployment
GitHub Pages: https://ejbrinker.github.io/e-plantShopping/

## Project Structure
- Landing page (App.jsx)
- About Us page (AboutUs.jsx)
- Product listing page (ProductList.jsx)
- Shopping cart page (CartItem.jsx)
- Redux cart slice (CartSlice.jsx)
- Styling (App.css)

## Technologies Used
- React
- Redux Toolkit
- Vite
- GitHub Pages
